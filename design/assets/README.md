# design/assets — finalized assets

The assets actually imported by `apps/`. Generated ones live one folder up in `generated/` — when we pick a final, it gets copied or symlinked here with a clean name.

Naming convention:
- `wordmark.png`, `wordmark-t.png` (transparent)
- `app-icon.png` (1024x1024)
- `favicon.png` (32x32 or 256x256)
- `og.png` (1200x630)
- `hero-scene.png`
- `agent-<species>.png`
- `agent-<species>-working.png`
- `agent-<species>-down.png`
- `agent-card-<species>.png`
- `human-chibi-base.png`
